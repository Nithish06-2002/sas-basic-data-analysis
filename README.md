# SAS Basic Data Analysis (Academic Version)

This repository contains SAS code developed using the SAS Academic version (SAS Studio).  
Only code is uploaded due to academic environment limitations.

## Dataset Used
- **SASHELP.CLASS** (built-in SAS practice dataset)

## Analysis Performed
- Viewing raw data using PROC PRINT
- Sorting data by categorical variable using PROC SORT
- Frequency analysis for categorical variables using PROC FREQ
- Descriptive statistics for numeric variables using PROC MEANS
- Distribution analysis using PROC UNIVARIATE (histograms)
- Group-wise summary analysis using PROC SQL

## Key Concepts Covered
- Library and dataset handling
- BY-group processing
- Categorical vs numeric data analysis
- Graphical representation of data
- SQL-based aggregation in SAS
